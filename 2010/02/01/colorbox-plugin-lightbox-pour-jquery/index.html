ColorBox, plugin lightbox pour jquery

01/02/2010

C'est pas les plugins lightbox basés sur jQuery qui manquent sur le marché. Le principe de la lightbox est d'afficher une fenêtre en surimpression (de la page active) avec pour contenu différents médias : images, vidéos, textes, formulaires...

La lightbox est très appréciée des concepteurs car elle permet de procéder à certaines configurations ou téléchargerments de fichiers sans quitter la page active. Cette fonctionnalité est une sorte de terrain d'entente entre les clients et les intégrateurs. Son utilisation n'est pas toujours du meilleur effet, mais elle remplace, de manière plus conforme aux règles de l'utilisabilité, la célèbre popup.

Pour éviter l'ouverture de nouvelles fenêtres, les intégrateurs pourront utiliser ColorBox. Respectant les règles du W3C, légère, gérant le préchargement d'images, les fondus, permettant les callbacks (...), cette fonctionnalité peut se révéler un solide avantage dans la réalisation de ses pages.

Découvrez ColorBox - A light-weight, customizable lightbox plugin for jQuery 1.3 and 1.4.
