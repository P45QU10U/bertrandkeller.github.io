IE-CSS3, support CSS3 pour IE 6, 7 et 8

15/06/2010

Voici un script pour interpréter les commandes de coins arrondis disponibles en CSS3

Comment ça marche ?

If you're viewing this page in Internet Explorer, some of the elements have been rebuilt by the script in Vector Markup Language (VML), an IE-specific vector drawing language. VML supports things that are missing from IE's CSS implementation like rounded corners and blur effects.

Pour les différents IE concernés, certains éléments sont reconstruits en VML, langage spécifique à IE. Le script permet seulement des générer les coins arrondis et les effets de flous.

On est bien loin de l'interprétation des toutes les propriétés CSS3 ; seuls sont disponibles les arrondis et quelques effets de flous. Cependant, dans un processus d'intégration, ces propriétés peuvent être amplement suffisantes.

En effet, gagner du temps sur la découpe de coins arrondis de quelques fenêtres de la page peut apporter bonheur et bien être. Imaginer la floppée de boutons arrondis sur vos formulaires ou encore les pieds de vos tableaux ; tous de largeurs différentes.
