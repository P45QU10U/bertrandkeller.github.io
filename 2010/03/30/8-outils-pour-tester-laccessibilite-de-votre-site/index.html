8 outils pour tester l'accessibilité de votre site

30/03/2010

L'accessibilité désigne le principe de pouvoir accéder aux contenus d'un site quelque soit la manière de naviguer sur le web. Par erreur, on estime que cela ne concerne, en pratique, que les personnes présentant un handicap (lourd) ; pourtant, la problématique de contenus accessibles s'étend à un public plus large. Pour mesurer la conformité de son site, il est nécessaire de le faire manuellement en s'aidant de divers outils existant sur le marché.

Voici une liste de 8 outils permettant de mesurer l'accessibilité de ses sites :

  • Color Blindness Simulator
  • Juicy Studios Image Analyzer
  • Firefox Accessibility Extension
  • Test and Improve Readability
  • LinkPatch
  • WAVE
  • AccessColor
  • aDesigner

Tester l'accessibilité, c'est tester la structure du code mais aussi les contrastes, les couleurs, l'intelligibilité des textes...
