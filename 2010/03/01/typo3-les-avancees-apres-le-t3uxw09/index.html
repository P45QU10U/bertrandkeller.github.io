TYPO3 : les avancées après le T3UXW09

01/03/2010

Le T3UXW09, la réunion dans le chalet pour faire avancer l'utilisabilité, que c'est-il passé ?

  1. Amélioration du Page Module, nettoyage des TCEforms et nouvelle recherche Backend :: refonte de l'interface d'édition des pages (sans templaVoila)
  2. Nouvel Introduction package, amélioration de la skin :: refonte du module d'installation
  3. Structuration des feuilles de style, nouvelle Skin API, ajout de sprites pour les icônes :: séparation intégrale des CSS
  4. ExtJS-ify du Backend TYPO3 :: refonte des événements JavaScript
  5. Mise en forme de ExtJS dans le Backend TYPO3 et amélioration mineures :: cohérences de l'apparence des fonctions ExtJS

Du beau travail, un nettoyage de printemps alors que les versions 4.x sont certainement en fin de vie. La dernière version 4.4 qui intégrera toutes ces améliorations sera donc plus propre et certainement plus performante.

La migration d'une version 4.4, à une version 5.x sera certainement facilitée.
