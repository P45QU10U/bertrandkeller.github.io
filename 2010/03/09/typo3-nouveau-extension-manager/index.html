TYPO3 : nouveau plugin extension manager

09/03/2010

On reste sur TYPO3, on reste sur la même dynamique de ce début 2010 : la refonte graphique des fonctionnalités principales du CMS.

Comme pour la gestion de pages hier, ou encore l'assistant d'installation ; le module de gestion des extensions vient d'avoir un petit relooking. Désormais, il est possible de classer les extensions en fonction de leurs spécificités, de faire des recherches...

Encore un fois, nous observons un beau travail de la communauté avec la reprise de modules historiques. Le fonctionnement est conservé, le travail se porte principalement sur la gestion graphique et l'affichage. Il s'agit donc d'ne vision interface de l'administration dont la philosophie et le code pourront être repris pour d'autres modules dans le futur.
