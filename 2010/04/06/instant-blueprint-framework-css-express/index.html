Instant Blueprint, FrameWork CSS express

06/04/2010

Je cherche, je cherche, je ne trouve pas le rapport entre le FrameWork Instant Blueprint, et le FrameWork Blueprint ; je ne vois pas le rapport. Si Bluprint est un FrameWork très complet pour développer des sites web, Instant Blueprint est une interface web permettant de disposer de fichiers de bases, pour débuter son site, de manière express.

Le principe d'Instant Blueprint se décline en 5 points :

  1. Indiquer le nom du projet
  2. Sélectionner le DOCTYPE et la librairie JavaScript
  3. Indiquer tous les tags d'id désirés
  4. Choisir une extension pour le fichier index
  5. Et euh... soumettre le formulaire

Voilà, un zip avec tout dedans, on gagne dix minutes pour un truc qu'on aurait pu faire à la main. Pratique, pas pratique ; je ne sais pas. Toujours bon à prendre en tous les cas.
