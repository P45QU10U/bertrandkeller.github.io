10 principes en or pour le développement d'applications web

02/04/2010

Les sites web ! Ha, les sites web ! Mais c'est pour les vieux, les sites web, ce qui compte aujourd'hui, c'est les applications web. Pffff, les applications web, je sais pas ce que c'est, mais c'est le présent des gens Hip. Parole de possesseur d'iPad.

Voici, les 10 principes en or pour le développement d'applications web par Fred Wilson :

The 10 Golden Principles of Successful Web Apps from Carsonified on Vimeo.

  1. Rapidité
  2. Utilité immédiate du service
  3. Le logiciel est le média
  4. Le moins est plus
  5. Programmable (par des développeurs)
  6. Facile d'adoption (par des développeurs)
  7. Architecture REST
  8. Visible dans les moteurs de recherche
  9. Propre
  10. Ludique

Encore une fois, on a l'impression qu'une Web App doit ressembler à une interface de jeu vidéo. Pourquoi pas ? Mais c'est quand même l'utilité d'un service, la possibilité de le modifier, la cohérence de son architecture qui feront son succès.
