PrimerCSS, obtenir une feuille de style CSS pour démarrer

06/01/2010

Coder du CSS + HTML est fastidieux, à peu près toujours les mêmes lignes de codes, mais pas tout à fait. La forme étant séparée du fond, il est possible d'opter pour la rédaction de sa page HTML et ensuite seulement de décliner les styles en CSS.

En effet, malgré toutes les solutions du marché en ce qui concerne le développement rapide de gabarits, la problématque sémantique nécessite quelques tortures de cerveau. La sémantique consiste à donner à ses identifiants et classes CSS des noms assez parlant pour définir des zones de contenu.

Exemples faciles à comprendre : menu pour le menu, contenu pour le contenu, titre principal pour le titre principal. Parfois, c'est quand même plus compliqué.

Primer CSS est une aide à la rédaction des classes pour démarrer la déclinaison des feuilles de styles en CSS. Un fois un gabarit HTML décliné, un petit copier/coller dans Primer CSS et se retrouve disponible une feuille de style minimale avec toutes les classes de la page HTML précédente. Ça permet de gagner un peu de temps.

L'idée est bonne au départ, mais le résultat n'est pas très probant. Le rendu donne les classes principales de la page HTML et pas grand chose de plus. Pour les doubles classes, les héritages, je n'ai pas vraiment vu de résultats. Sûrement pratique pour débuter sa feuille de style mais pas indispensable. Dépend en grande partie de sa façon de coder à mon avis.

Essayer PrimerCSS.
