ProCSSor, outils de formatage de vos CSS

15/09/2010

Je vois rien, là, c'est quoi ce code ? Une CSS compressée ; je fais comment pour la récupérer ? plus de 15 ans d'expérience dans les CSS et encore obligé de ré-indenté tout à la main ?

Non car il existe sur le web ou dans ton éditeur de CSS des fonctions te permettant de reformater tes sélecteurs. A ce propos, va voir ProCSSor, le reformateur le plus complet actuellement.

Il te permettra à partir de code collé, d'un fichier ou encore d'un URL (d'un feuille de style) de recomposer les lignes selon ta convenance :

  • Disposition des sélecteurs
  • Disposition des propriétés
  • Indentation des propriétés en colonne (toutes propriétés à la même distance) ou indentées

La disposition en colonne n'est pas idéale car les propriétés passent sur la ligne de dessous en cas de sélecteurs trop long, mais en cas de reprise de projet ou d'un besoin d'analyse d'un code, proCSSor est ton ami.
