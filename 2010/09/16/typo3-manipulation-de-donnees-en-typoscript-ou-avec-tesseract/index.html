TYPO3 : manipulation de données en TypoScript ou avec Tesseract

16/09/2010

Deux informations intéressantes sur la possibilité de manipuler des données de la base de données pour les restituer sur les pages de vos sites.

Manipulation de données en TypoScript

Sebastiaan Delonge nous explique que la manipulation de données de la base en TypoScript est limitée par défaut. Il nous donne un moyen de passer outre cette restriction pour aller taper des données en base sans passer par la création d'un plugin.

Il a attendu d'avoir une confirmation de l'équipe de sécurité à propos de l'existence de cette restriction pour remettre en ligne son article.

Lire TypoScript bypassing database table restrictions.

Tesseract

La société suisse Cobweb, spécialisée en TYPO3, présente le projet Tesseract permettant de récupérer des données de la base pour les restituer grâce à un éléments de contenu dans une page.

A short screencast highlighting the base features of the Tesseract Project for TYPO3. A suite of extensions designed to manipulate data easily. In this example we will output a list of BE users in the TYPO3 frontend in just a couple of minutes, without doing any development.

Tesseract introductory screencast from cobweb|tv on Vimeo.
