Drupal est un CMS pratique et complet, il permet d'éditer plusieurs types de contenus sur le web ; pour vous le prouver, Glen édite une liste de sites, type de sites ou services que Drupal peut permettre de cloner.

  1. Digg Clone
  2. Blogs
  3. Portails d'actualités
  4. Sites d'utilisateurs robustes
  5. Sites informationnels
  6. Twitter Clone
  7. Site de stockage de fichiers
  8. Flickr Clone
  9. Delicious Clone
  10. YouTube Clone
  11. Amazon Clone
  12. Tumblr Clone
  13. Blogger Clone
  14. Réseau de blogs
  15. Agrégateur d'actualités
  16. Friendfeed

Donc il existe effectivement les fonctionnalités dans Drupal pour cloner les particularités des sites cités ; néanmoins, il s'agit de web, il est possible de les recoder de manière spécifique par soit même. De plus, il n'est pas dit que ces fonctionnalités répondent à tous les besoins de performance nécessaires à certains projets.

Drupal offre un panel de modules plutôt fourni qui permet de faire évoluer un site éditorial et social de manière sereine ; surtout si vous n'avez pas de compétences poussées dans le développement.

Si vous ouvrez un site d'actualités en ligne, premièrement vos lecteurs pourront bénéficier d'un espace personnel fonctionnel ; deuxièmement, le gouvernement subventionnera votre journal à hauteur de 200 000€.
