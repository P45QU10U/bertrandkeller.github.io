Le site TYPO3 dispose d'un nouvel outil permettant d'indiquer ses références web réalisés grâce à cette technologie : "Sites made with TYPO3: Beta".

Les fonctionnalités de cette extension sont assez poussées puisque il est possible de remplir un certain nombre de critères sur lesquels une recherche peut être effectuée ; les critères : nom du site, URL du site, date de mise en ligne, description, tags libres, liste des extensions utilisées, (...), pays, nombre de pages...

Si on regarde le bloc listant les sites par pays, on remarque que la France dispose d'un faible rang (une dizaine de sites tout au plus) ; effectivement peu d'agences ont dû prendre connaissance de ce nouvel outil. Je me propose de lister les agences web francophones les plus réputées afin d'avoir un meilleur aperçu des réalisations dans cette langue.

  • Infoglobe - Québec
  • Smile - Levallois-Perret
  • Adaltas - Thise, Vincennes
  • CYO - Roubaix
  • Bleu-Roy- Marseille
  • Mind2machine- Marseille
  • Archriss - Marseille / Aix en provence / Miramas
  • Onext - Nantes
  • Novae communication - Pau
  • Ameos - Mommenheim
  • Oblady - Amiens
  • Artifica - Paris
  • Hyppocampe - Paris
  • Mediasmart - Paris
  • Smart Agence - Paris
  • Kleegroup - Le Plessis-Robinson
  • One Pixel Studio - Genève
  • Cobweb - Genève
  • Eosgarden - Lausanne
  • Modulis - Montréal

Certaines de ces sociétés seront représentées à l'université TYPO3 qui aura lieu à Annecy du 30 juin au 4 juillet 2008 : "le programme des conférences".
