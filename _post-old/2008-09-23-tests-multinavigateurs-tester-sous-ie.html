Tandis que la campagne contre ce maudit navigateur qu'est IE6 bat son plein, de nombreuses solutions sont mises en ligne pour tester un site, rapidement, sous les différentes versions d'Internet Explorer.

Le web doit évoluer, le navigateur IE6 doit être abandonné, il ne respecte pas les standards. Trop d'exceptions sont mise en places pour l'affichage de pages web. Les utilisateurs d'IE6 obligent les développeurs à consacrer une partie de leur temps spécifiquement pour ce navigateur.

Pour preuve de la mobilisation de codeurs pour l'abandon d'IE6, visitez des sites comme Stupid IE ou encore Save the developper.

Pour ceux qui sont dans l'obligation de continuer à tester sous tous les navigateurs de Microsoft, plusieurs solutions existent pour disposer de toutes les versions d'IE sous un même système d'exploitation.

Multiple IE et Internet Explorer Collection
Installeurs permettant de disposer de plusieurs versions d'IE en version standalone. Internet Explorer Collection semble plus complet puisqu'il comporte toutes les versions d'IE en allant de la 1.0 jusqu'à la 8.0 en version beta, alors que Multiple IE n'installe IE que jusqu'à la version 6.

IE tester
Navigateur permettant de tester toutes les versions d'IE dans différents onglets.

Pour vous inciter à changer de navigateur pour améliorer les performances de l'affichage de vos pages web (contenant du javascript) : Navigateurs : une comparaison des performances JavaScript.
