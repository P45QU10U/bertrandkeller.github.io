Petite recherche pour la configuration du code des div (container) qu'il est possible d'ajouter sur la version du Rich Text Editor de la version 4.2 de TYPO3.

Code

 // Permet l'insertion de balises en dehors de balises p lib.parseFunc_RTE.externalBlocks = table,blockquote,ol,ul,hr,div lib.parseFunc_RTE.externalBlocks { 	div { 		// Indentation du code HTML 		stripNL = 1 		// Repasse du parser HTML sur le contenu interne à la div 		callRecursive = 1 	} }

Ces lignes de code permettent d'avoir une balise div en dehors d'une balise p (sinon une balise p est rajoutée autour de la div lors du rendu) et aussi de faire repasser le Parser à l'intérieur du contenu de cette même div (sinon le texte à l'intérieur de la div est affiché sans balises p lors du rendu).
