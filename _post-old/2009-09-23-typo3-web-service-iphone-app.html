TYPO3 web service

L'extension TYPO3 web service permet de créer, lire, éditer et effacer des données sur TYPO3 à l'aide d'une application externe.

The new TYPO3 web service - developed by naw.info - enables you to create, read, edit, and delete data in TYPO3 through an external application by using the extension "typo3 webservice".

TYPO3 iPhone App

A titre d'exemple, une application iPhone peut permettre à un utilisateur de gérer les actualités d'un de ses site sous TYPO3.

As one example of practical usage of the TYPO3 web service extension, naw.info developed an iPhone App that enables users to maintain TYPO3 news
