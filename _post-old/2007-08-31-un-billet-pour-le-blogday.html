Le 31 août, un billet pour le blogday, 5 petits liens pour faire découvrir le web. Comme je suis un working boy, j'ai pas vraiment eu le temps de contacter les propriétaires des sites suivants, que ceux-ci ne m'en tiennent pas rigueur.

Le blog des Vins LELIEVRE
Un village dans lequel, j'ai passé un partie de mon enfance ; un famille dans laquelle, j'ai passé quelques week ends ; le blog de la famille Lelièvre est pour moi un exemple de numérisation du territoire.

Ecrire pour le web
Avec ou sans le Bic rouge, "écrire pour le web" est écrit pour le web. J'aime beaucoup ce dernier billet sur le splash screen.

Christian Fauré
Pour ses enregistrements de Bernard Stiegler.

NSI solution
Une réflexion appronfondie sur le l'entreprise 2.0, comment expliquer à son patron que naviguer sur le web n'est pas une perte de temps.

Jaiunblog
Je ne lis pas ce blog mais vous pouvez le faire si vous voulez.
