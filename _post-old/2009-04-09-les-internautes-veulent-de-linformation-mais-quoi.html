Muriel a essayé de nous présenter sous quelle forme présenter le contenu : Les internautes veulent de l’information. Mais sous quelle forme ?

Après avoir étudié la présentation des informations, nous pouvons étudier quel type de contenu mettre en ligne. Dans le billet de Sylvain Guéguen, les propositions sont assez spécifiques aux sites de marques : Stratégie de marque : Comment produire du contenu utile ?.

Le défi pour les marques aujourd’hui sur le web est de pouvoir se faire une place dans l’information que nous consommons. Il devient donc nécessaire pour elles de légitimer d’un point de vue utilisateurs, leurs initiatives sur le web sous peine de passer inaperçus. C’est ce que l’on appelle l’économie de l’attention : une expression très en vogue chez les communicants de mon espèce.

Alors comment légitimer cette présence ? Un seul mot d’ordre : « be relevant » : être pertinent, être approprié, etc. Finalement s’adapter à nos attentes, voir les surpasser.

Il met en avant trois manières de communiquer pour une marque :

  1. En apportant des Conseils & infos pratiques
  2. En offrant des moyens pour se divertir : Entertainment
  3. En orientant les utilisateurs vers une solution adaptée : Rôle de connecteur

Si on devait généraliser ces conseils pour l'ensemble des sites, on reformulerait de la sorte :

  1. Définir le(s) thème(s) du site et éditer des ressources pertinentes pour la communauté (liée à ce thème)
  2. Permettre aux utilisateurs de rebondir vers d'autres thèmes (liés directement ou indirectement)
  3. Connecter les utilisateurs par la mise en place d'outils de communication "sociaux"

Tout cela se résume très facilement. Les ressources, les actualités, les jeux, les forums, les commentaires... représentent des types de contenu.

Un site doit pouvoir exploiter les avantages de chacun de ces types de contenu en fonction du public visé.
