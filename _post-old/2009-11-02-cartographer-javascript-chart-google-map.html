Cartographer.js est une librairie JavaScript qui permet de superposer des graphiques (charts) sur des cartes provenant du site Google Map.

Cette librairie propose divers types de représentations :

  • Chloropleth - zones colorées
  • Cluster - cercles
  • Pies - camembert

La visualisation passe par une autre librairie JavaScript nommée Raphaël.

Le placement des graphiques peut s'effectuer en absolu en utilisant les latitudes et longitudes. Il semble aussi possible de colorer certaines zones administratives en utilisant leur label (exemple : US-OR ou l'Oregon).
