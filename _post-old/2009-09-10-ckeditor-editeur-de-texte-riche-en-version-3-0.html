Il existe, sur ce blog, un billet à propos des éditeurs de texte, Rich Text Editor (RTE). Je liste les éditeurs de contenu les plus intéressants du marché. Ma petite contribution permet de gagner quelques heures dans sa recherche.

Depuis le 15 janvier 2009, FCKeditor a changé de nom pour se prénommer CKeditor et propose le téléchargement de sa version 3.0.

Quoi de neuf, sur la nouvelle version de CKEditor 3.0 ?

  • Performance de folie
  • Accessibilité
  • Nouvelle interface vraiment cool
  • Nouvelle API JavaScript
  • Pleins d'autres trucs trop long à expliquer...

Effectivement, cette nouvelle version de l'éditeur semble très alléchante.
