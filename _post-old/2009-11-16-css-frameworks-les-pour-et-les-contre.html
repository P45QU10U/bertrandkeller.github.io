Les FrameWorks

Un FrameWork CSS est constitué de feuilles CSS contenant des classes prédéfinies. Leur code permet de décliner un graphiste rapidement en respectant les règles de mise en page web : grilles, définition typographique...

Liste de FrameWorks CSS

  • Blueprint CSS
  • Elements
  • YUI 2 : Grids CSS
  • Logic CSS
  • Elastic CSS

Les avantages

  • Conformité multi-navigateurs
  • Développement accéléré
  • Cohérence
  • Feuilles propres

Les désavantages

  • Présence de code inutile
  • Manque de liberté
  • Incohérence sémantique des classes CSS

Les FrameWorks CSS, pour tout dire, ça dépend. Il vaut mieux avoir le sien ou en trouver un qui correspond à sa façon de coder.
