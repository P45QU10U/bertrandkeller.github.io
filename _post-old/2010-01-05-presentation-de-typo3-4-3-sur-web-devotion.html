Christian Bellet, de Typo3Journal, présente la nouvelle version de TYPO3 en version 4.x sur le site Web Devotion. 4 pages pour présenter diverses évolutions de TYPO3 4.3

  • structurel car la version 4.3 est un pas en avant vers la version 5.0
