Comme le dit Picasso, l'artiste copie les oeuvres, le génie les vole. Avec CSS Bake, vous avez la possibilité de vous informer sur les graphismes web les plus réussis.

CSS Bake classe les ingrédients indispensables pour la réalisation de vos sites web. Les catégories disponibles sont :

  • 404 Page
  • Archives
  • Fonds de pages
  • Commentaires
  • Pieds de pages
  • Formulaires
  • Entêtes
  • Navigations
  • Paginations
  • Billets de blog
  • Colonnes contexttuelles

Non seulement, ce site vous propose de magnifiques exemples de mises en page, mais aussi liste toutes les pages et tous les éléments à décliner pour votre site web.
