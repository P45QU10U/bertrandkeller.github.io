On commence à connaître le principe des FrameWorks CSS, ces bases de codes permettant de décliner des gabarits de manière sûre et rapide. Regardons un peu du côté d'EZ-CSS, un FrameWork très léger qui permet de créer diviser des zones de contenus à volonté.

Le principe est de copier/coller des blocs prêt à l'emploi dans des containers. J'ai une zone de contenu que je veux diviser en 3, je copie le code pour 3 colonnes et je le colle dans mon container et ainsi de suite à l'infini.

Le fichier CSS ne comprend que le code pour diviser une zone en x colonnes et les faire flotter dans le bon ordre. L'auteur a ajouté quelques lignes pour la compatibilité multi-navigateurs (IE) et le tour est joué.

EZ-CSS se montre très pratique pour la division rapide des zones de contenus de sa page. Justement, EZ-CSS propose une version encore plus basique spécialement dédiée au prototypage.

C'est Jean-Paul du marketing que vous pouvez impressionner avec un site plus vrai que nature. Enfin, faites gaffe quand même, il va croire que le boulot est terminé.

EZ-CSS est un FrameWork a montrer aux quarantenaires développant encore en tableau et à utiliser pour les mises en page à réaliser rapidement.
