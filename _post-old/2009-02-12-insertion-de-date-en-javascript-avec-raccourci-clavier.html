Unobtrusive JavaScript date-picker widgit, une petite fonction JavaScript permettant l'insertion de date dans un champ texte, propose une navigation clavier simple et élaborée.

  • Flèches : Navigation sur les jours
  • Page haut : Mois précédent
  • Page bas : Mois prochain
  • Ctrl + page haut : Année précédente
  • Ctrl + page bas : Année suivante
  • Espace : Date courante
  • Echap : Fermer le calendrier
  • Retour chariot : Sélectionne la date active et ferme le calendrier

On s'empresse de la tester, cette fonctionnalité respecte les DHTML Style Guide Working Group (DSGWG).
