Pour le développement d'interfaces web destinée au mobile, on connaît plusieurs librairies JavaScript très poussée. Par exemple jQuery Mobile propose des exemples de fonctionnalités qu'il est possible d'intégrer à son site. Sur ce site, le graphisme (présenté) ressemble fortement à celui d'une application native ; néanmoins, il est possible de le personnaliser selon ses besoins et, voire, de réutiliser des fonctions pour un site web, dit classique.

Le FrameWork mb.ile permet aussi de mettre en forme des applications pour le mobile. Si on fait un tour sur les pages proposées, on trouve des exemples d'intégration très intéressants : mb.ile.

  • Menu - menu déroulants, big menu, menu recherche avancée...
  • Video player - lecteur vidéo en fond de zone de contenu
  • Vertical slider - menu déroulant vertical de style carrousel
  • Extruder - menu latéral fixe, rétractable
