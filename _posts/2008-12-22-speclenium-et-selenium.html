Specular

Specular est une suite d'outils et de librairies permettant de réaliser des tests d'accessibilité du point de vue utilisateur.

Speclenium

Speclenium est un service (daemon) dont Selenium tire avantage pour l'exécution de fonctionnalités définissant les tests d'accessibilité.

Selenium

Mais alors qu'est-ce que Selenium ? Selenium est le nom d'un projet dans lequel est regroupé un certain nombre outils de test.

Selenium IDE est un univers de développement disponible par une extension Firefox. C'est fait pour coder des tests pour les interfaces utilisateurs.
