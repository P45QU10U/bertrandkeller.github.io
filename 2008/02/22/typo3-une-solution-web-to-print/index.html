TYPO3 une solution Web-to-Print

22/02/2008

TYPO3, une solution web vers un export impression. Je reprends l'actualité du blog de Daniel Pötzinger qui parle de cette extension éditée par Timo Schmidt. Je vous donne la traduction en français de cet article en anglais, lui même tirer d'une version en allemand.

L'idée de départ est de réemployer le contenu édité sur une interface web pour en exporter une version imprimable, le principe de fonctionnement est celui-ci :

  • Le rédacteur compose une sélection d'éléments de contenu (blocs TYPO3)
  • Cette sélection hiérarchisée peut être stocker dans l'interface
  • Le plugin permet d'exporter cette sélection d'éléments de contenu dans différents formats avec une première étape en XML : LaTeX, InDesign (Adobe TaggedText)

L'auteur de cette extension espère par la suite, avec la même méthode, exporter les contenus TYPO3 vers d'autres applications comme Open Office, MsOffice, QuarkExpress, etc... Inutile de se demander pourquoi Timo passe son temps à la montagne, que de minutes gagnées en rédigeant son contenu une et une seule fois, le tout en HTML.

Imaginons seulement une organisation productrice de contenus, à peu près la quasi majorité des organisations ; désormais, elle pourra éditer son contenu sur une interface ouverte à un certains nombre d'utilisateurs, soumettre ce contenu, pour le faire valider et évoluer, et enfin sélectionner les briques nécessaires pour la communication de documentations ou de catalogues.

Cette évolution n'est que la concrétisation des possibilités de formatage des contenus. Sa réussite est néanmoins dépendante de la qualité de rédaction (écriture) de ce même contenu ; il pose aussi le problème du contrôle des versions des ressources qui sont loin d'être identifiables facilement sur le web.
