Le macaron du meilleur blog TYPO3 de l'année est attribué a...

01/02/2008

Frank Nägler gagne le titre de meilleur blog TYPO3 de l'année pour son site typo3weblog : http://typo3weblog.de/.

Le concours était organisé sur le site Fachinformatiker. Pour la petite histoire, tous ces rédacteurs éditent sous le moteur WordPress, le premier dans le classement à tourner sous le plugin mythique tt_news est Daniel Pötzinger avec TYPO3 Blog.

Toute une année pour placer un blog francophone dans le concours 2008. Pour le moment le mieux placé est typo3journal.
