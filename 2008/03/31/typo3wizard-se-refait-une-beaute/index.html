Typo3wizard se refait une beauté

31/03/2008

Typo3wizard, le site présentant la plus grande bibliothèque de snippets TYPO3, vient de refondre son site. La couleur verte est toujours d'actualité, on retrouve la structure du site précédent, mais le contenu est nettement plus lisible.

Dans les pages intérieures, la liste des snippets, classés par thème, est présentée sous la forme d'un menu situé au dessus du contenu ; celui-ci facilite grandement la navigation (La navigation par thème est possible grâce à la navigation contextuelle à gauche ou encore en passant par le fil d'Ariane).

Le code TypoScript est présenté dans une division sur fond blanc avec un sur-lignage (highlighting) différenciant les deux parties des déclarations. Il est possible de copier directement le code dans un éditeur sans rajout de caractères (comme c'est agréable).

En bas de page, se situe, parfois, un lien vers un exemple de rendu du code (pratique pour les menus) ; son caractère éventuel aurait demandé de le mettre mieux en valeur.

Typo3wizard est un site qui manque sérieusement d'animation. Sa refonte graphique rend son contenu plus lisible mais ne suffira pas à inciter les visiteurs à participer de façon plus active. Je remercie Helmuth Antholzer et Tobias Erlacher pour avoir réalisé ce site et espère qu'ils parviendront à lui donner une dimension plus importante.

Bertrand Keller ; une espèce rare ; un défenseur du point-virgule
