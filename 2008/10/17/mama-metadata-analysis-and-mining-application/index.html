MAMA : Metadata Analysis and Mining Application

17/10/2008

MAMA, pour Metadata Analysis and Mining Application, est un moteur de recherche qui permet en quelque sorte d'analyser la structure du Web en indexant tout un lot d'informations sur les pages qui le constituent. Cela donne ainsi une vision d'ensemble des technologies Web actuellement utilisées (CSS, HTML, XHTML, balise spécifique...).

Source : Génération NT - MAMA : Opera étudie la structure d'un Web assez peu W3C.

Cet outil est surtout destiné aux développeurs de navigateurs qui pourront voir comment les technologies qu'ils implémentent sont adoptées selon les pays.

Pour des intégrateurs, ces données sont instructives mais pas forcément très utiles à l'échelle du web. En revanche, étudier la qualité d'un corpus donné (un intranet par exemple) peut se révéler très intéressant pour le suivi de ses sites web.

Pour en savoir plus sur MAMA, lire Brian Wilson sur le site d'Opéra, MAMA ; ou le blog Helion Prime, Understand Web with MAMA.
