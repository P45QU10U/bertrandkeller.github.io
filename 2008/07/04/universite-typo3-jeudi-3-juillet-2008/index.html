Université TYPO3 jeudi 3 juillet 2008

04/07/2008

"Best practises" pour le développement d'extension

En tant qu'expert TYPO3 Pierre Dudoret est intervenu à propos des bonnes pratiques à appliquer pour le développement sur TYPO3. Sa présentation est certainement à retravailler pour présenter de façon plus cohérente une façon de coder qui serait propre à TYPO3. On aura juste retenu quelques conseils bien utiles, à savoir : lire les notices qui sont sur les différents sites, utiliser les classes présentent dans t3lib_db pour se servir des fonctions intégrées du CMS, lire les manuels et lire les documentations.

Retour sur les performances

L'intervention sur le RealURL étant annulée, nous sommes revenu sur l'entrevue à propos des performances que nous avions eu la veille : prévoir des tests de montée en charge ; séparer le serveur web et la base de données ; passer par du load balancing si nécessaire ; activer les caches TYPO3, de PHP, de mysql (en fonction du nombre de connections) ; désactiver certaines extensions en production...
