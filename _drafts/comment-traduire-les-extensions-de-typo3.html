La traduction des plugins sous TYPO3 est une question intéressante, exactement celle sur laquelle je suis en train de plancher. La réflexion est à son début, pour le moment, nous avons (je suis pas tout seul) cherché les plugins qui pourraient nous servir :

Language File Editor

http://typo3.org/documentation/document-library/extension-manuals/lfeditor/2.6.2/view/1/2/
Autre plugin pour éditer les constantes par langue.

Rename Form-Field-Label via PageTS

http://typo3.org/documentation/document-library/extension-manuals/maja_labelrename/0.0.4/view/1/2/#id3554027
Cette extension permettrait de surcharger des variables contenu dans les fichiers xml. Une possibilité supplémentaire d'édition.

http://typo3.org/extensions/repository/view/a4n_edit_locallang/1.1.1/
