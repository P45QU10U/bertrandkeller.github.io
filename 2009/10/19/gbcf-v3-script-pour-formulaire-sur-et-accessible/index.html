GBCF-v3 Script pour formulaire sûr et accessible

19/10/2009

GBCF est un script PHP permettant de vérifier la validité d'un formulaire HTML. Les formulaires sont des outils puissants pour collecter des informations sur les usagers d'un site ; seulement les informations doivent être remplies de manière satisfaisante afin d'être exploitées et ne doivent pas être polluées par du spam.

Ce script PHP permet à la fois de vérifier les informations insérées dans un formulaire, mais propose aussi des possibilités de retour d'information pour une meilleure compréhension par l'utilisateur. Quelques fonctionnalités en anglais :

  • Easy Usage Terms
  • Easy Setup Steps
  • Automatic Accessibility
  • Hardened Security
  • Persistent Validity
  • Language Support
  • Inherent Intelligence
  • Error Management
  • Smart Feedback
  • Back Button Gone
  • Independent Function
  • Input Highlights
  • Blocked/Error Stats
  • Anti-Flood Control
  • Myriad Advanced Settings
  • Admin Lockdown Mode
  • Thank You Option
  • Excellent Support
