Tuner son WordPress avec SEO Super Comments

02/10/2009

La lutte pour l'amélioration de son référencement est implacable. Malheureusement, elle se traduit souvent par une action de tuning de son site.

Mais si bien sûr, vous avez déjà vécu ça, quand le responsable (au marketing) du site officiel de votre société est venu vous voir en vous disant :

* Il paraît que tu as un blog, j'aurai besoin que tu me fasses une petite vérification référencement du nouveau site corpo, celui qui va bientôt sortir.

+ Oui bien sûr, si tu veux...

* J'aimerai apprendre tous les secrets du référencement

+ C'est-à-dire que... oui... mais enfin, vous avez décidé quand de lancer ce nouveau site ?

A quoi se résume le soit disant SEO (Search Engine Optimization) de ce site ? A remplir quelques champs meta avec des mots clefs appropriés pour que le moteur classe le mieux possible votre page ; rendre certains liens crawlables ou non crawlables ; créer un compte google webmaster tools.

Ce n'est pas en rajoutant un aileron à une 2CV que vous pouvez obtenir les performances d'une Aston Martin. Le référencement se pense dès la conception du site principalement par la structuration des données. Par la suite une savante maîtrise de la qualité rédactionnelle (contenu de valeur) et du linking (avoir des amis importants et classes) permettra d'intéresser le web entier à votre site.

Dans la rubrique tuning de son blog, nous avons donc le plugin SEO Super Comments ; celui-ci permet d'attribuer une URL (adresse) à chacun des commentaires de votre site.

For each user comment we will create a new dynamic page on the blog, holding the actual comment information. This page will not actually exist in WordPress database, but we will create it dynamically using a WordPress plugin.

Quand le robot de google passe sur votre site, il va considérer chaque commentaire comme une page. Ce plugin permet ainsi de gonfler le nombre de pages du site ; d'augmenter de le poids d'un blog ; d'améliorer le trafic.

SEO Super Comments, une véritable extension de tuning.
