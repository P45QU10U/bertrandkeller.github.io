3 projets Open Source à suivre dont TYPO3

14/10/2009

Sur le site Royal Pingdom, Scott Nesbitt nous propose de suivre trois projets Open Source particulièrement innovants.

  • Piwik - analyse de trafic
  • OpenLaszlo - plateforme de développment de RIA
  • TYPO3 - CMS pour entrerprise

TYPO3 is easy to use and incredibly flexible. As well, TYPO3 is packed with dozens of features that enable you to quickly create and deploy a website. If you’re not a web designer or developer then Typo3 might just be the fastest and easiest way to build your web presence.

Je ne trouve pas les arguments très décisifs. On retiendra néanmoins la citation de TYPO3 dans cette liste. A noter que Pingdom propose un service de monitoring de performance pour votre site et que leur interface est très complet ; leurs conseils ne doivent pas être pris à la légère.
