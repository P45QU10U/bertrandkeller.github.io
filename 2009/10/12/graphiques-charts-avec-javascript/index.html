Graphiques (Charts) avec JavaScript

12/10/2009

La création de graphiques à partir de tableaux HTML devient plus facile grâce à des librairies de visualisation en JavaScript.

Le principe est simple : afficher un tableau sur une page HTML, ajouter les classes qui vont bien et appliquer une librairie de visualisation. En fonction des options choisies, la page affiche différents types de graphiques.

Ce procédé est bien pratique pour construire des tableaux de bord en deux temps, trois mouvements. Il suffit juste de remonter des données correctement formatées.

Voici un lien vers jQuery Visualize Plugin qui gère ce genre de tableaux, mais aussi d'autres librairies de graphiques en JavaScript :

  • jQuery Visualize Plugin
  • gRaphaël—Charting JavaScript Library
  • TufteGraph
  • Canvas 3d graph
  • MooChart
  • Bluff
  • RGraph

Grâce à elles, vous pourrez, par exemple, partager des données importantes sur l'avancée de vos projets avec toute votre équipe.
