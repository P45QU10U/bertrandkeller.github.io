jQuery : 5 nouveaux plugins de novembre

24/11/2009

La librairie jQuery vous passionne, ses plugins égaillent vos sites avec ses animations de grande classe. Ce mois-ci 5 nouveaux plugins jQuery qui vous permettront de mettre en forme vos pages.

  1. jQSlickWrap - faire suivre au texte le contour de vos illustrations
  2. jQuery Slider plugin (Safari style) - slideshow avec curseur de position à la mode Safari
  3. jQuery Shuffle Plugin - afficher du contenu dans un ordre aléatoire
  4. Flips - faire coulisser des blocs de texte
  5. jQuery Sprite Animation Plugin - animation d'images sprites
