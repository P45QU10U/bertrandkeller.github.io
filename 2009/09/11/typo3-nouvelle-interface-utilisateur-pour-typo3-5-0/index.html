TYPO3 : nouvelle interface utilisateur pour TYPO3 5.0

11/09/2009

Jens Hoffmann & Björn Brockmann l'avaient promis voilà l'interface du nouveau TYPO3 5.0. On passe du bleu azur au gris ; un peu terne mais plutôt classe.

On ne change pas trop la philosophie. Un menu à gauche avec l'arborescence, le contenu sur la droite. Au dessus, l'entête permet de naviguer dans les différentes fonctionnalités du CMS.

Un gros travail sur l'utilisabilité ; des icônes et une tooltip (boite de dialogue) associée pour l'information textuelle. On dira que l'interface est épurée, simplifiée afin de ne plus effrayer les utilisateurs à la première prise en main.

Authors: Jens Hoffmann, Björn Brockmann & Rasmus Skoldjan ©2009 TYPO3 Association

Un effort semble avoir été réalisé pour une édition Front End optimisée. Un autre moyen de faciliter l'édition de contenu par les rédacteurs.

Authors: Jens Hoffmann, Björn Brockmann & Rasmus Skoldjan ©2009 TYPO3 Association

Pas de révolution réelle dans l'interface ; nous restons dans une continuité avec les interfaces des versions précédentes. Premièrement, les innovations principales de TYPO3 5.0 se trouvent dans son moteur et pas dans son interface. Deuxièmement, les développeurs avaient promis la migration possible entre des versions 4.x et 5.x, cette interface est une preuve de compatibilité probable.
