Tim Berners Lee: Raw Data Now !

18/03/2009

Tim, c'est la papa du web. Monsieur Berners Lee a développé les adresses web, le Hypertext Transfer Protocol (HTTP) et le Hypertext Markup Language (HTML).

Le monde lui doit énormément avec son objectif de rendre son idée complètement libre sans brevet ni droits. Cette idée permet aujourd'hui la libre circulation des connaissances.

Pourtant, comme vous le savez, même si le web est ouvert, certaines données restent disséminées, enfermées dans des silos, et ne sont pas naturellement interopérables. D'où le travail de Tim sur ce qu'on appelle le web sémantique avec le "Linked Data" et maintenant le Raw Data Now!.
