jQuery : plugin ARIA

27/03/2009

WAI-ARIA (Web Accessibility Initiative - Accessible Rich Interactive Applications) est une spécification qui propose la possibilité de définir une description des rôles, des états et des propriétés pour les widgets personnalisés, de manière à ce qu’ils soient reconnaissables et utilisables par les utilisateurs de technologies d’assistance.

Pour définir ces rôles, états et propriétés dans le code HTML d'une page, il est nécessaire de passer par une librairie JavaScript.

jARAI est une extension jQuery permettant la gestion d'ARIA pour des interfaces riches. Une explication de son fonctionnement est disponible sur le site de Chris Hoffman : jARIA jQuery Plugin - Outstanding Elephant.
