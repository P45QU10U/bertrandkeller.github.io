jQuery : Zoom

27/03/2009

Si vous recherchez un code JavaScript pour permettre de voir plus en détail une image ; je vous conseille d'aller faire un tout sur le site Mind Project pour découvrir jQzoom.

Ce développement permet de réaliser différents types de Zoom :

  • Standard zoom
  • Reverse zoom
  • Zoom without lens and title
  • Custom positioning the jqzoom window
  • Fade out effect while hiding
  • Fixed IE6- select box bug
