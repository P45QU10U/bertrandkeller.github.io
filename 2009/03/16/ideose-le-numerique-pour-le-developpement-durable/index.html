Ideose : le numérique pour le développement durable

16/03/2009

C'est beau, ça fait rêver. Pierre GUILLOU crée Ideose après 6 années à la tête des activités AccessiWeb (accessibilité du Web : accès au Web par tous les internautes et par tous les modes de consultation).

L’accessibilité peut garantir à un propriétaire de site Web que son information en ligne sera lue par tous quelque soient les modes de consultation. Encore faut-il que les méthodes de développement et les solutions adaptées existent pour rendre accessible les nouveaux objets Web : le Web 2.0, les applications Web, les logiciels, les outils de gestion de contenus (CMS, blog…), les formats dits propriétaires (Word, PDF…), les animations Rich Media et les vidéos, les supports de consultation autres que l’ordinateur classique, les mobiles, la télévision numérique… Il faut donc passer à une nouvelle étape technologique : créer des solutions innovantes et accessibles pour les nouveaux usages Web.
