dygraphs JavaScript Visualization Library

10/12/2009

dygraphs est une librairie JavaScript pour la visualisation de données. Elle est dédiée à l'affichage de données denses.

The dygraphs JavaScript library produces produces interactive, zoomable charts of time series. It is designed to display dense data sets and enable users to explore and interpret them.

Les graphiques de la librairie dygraphs affichent la valeur des points (abscisses et ordonnées) que la souris survole. Il est possible de comparer la valeur de 2 courbes à une date donnée.

En bas à gauche des graphiques, se trouve un champ texte avec un chiffre. La modification de ce chiffre permet le lissage de chacune des courbes. Le rendu est donc moins précis mais permet d'avoir une appréciation de tendances.

Encore mieux, la définition d'un cadre à l'intérieur du graphique par un clic prolongé permet de zoomer sur les données. Un double clic réinitialise le graphique comme à son origine.
