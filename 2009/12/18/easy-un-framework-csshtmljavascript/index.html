Easy, un FrameWork CSS/HTML/JavaScript

18/12/2009

Des FrameWork CSS, on en parle, des FrameWorks HTML, on en voit passer, Des FrameWorks JavaScript, on en utilise un peu, mais alors des FrameWork CSS/HTML/JavaScript, ça ne court pas vraiment les rues, mais ça pourrait commencer à en intéresser certains.

Easy, est l'aboutissement d'un petit projet, dans lequel est proposé du code divisé selon trois couches : Struturel (HTML), Présentationnel (CSS) et Intéractif (JavaScript).

Comme c'est pratique, comme ça semble facile à manipuler. Je me procure un gabarit HTML de base disposant de la déclinaison de tous les éléments dont j'ai besoin ; je modifie ensuite ma feuille CSS pour la mise en forme du graphisme ; pour finir j'insère quelques classes CSS pour appliquer des événements pour l'intéraction.

Ce type de projet permet de décliner un gabarit propre sur une base de code solide. En effet, le code HTML est très complet et propose l'ensemble des éléments qu'on peut trouver sur une page. Sur cette base, il devient très facile de décliner son gabarit à sa manière en bougeant les zones de contenu, en en rajoutant des illustrations...

Le petit plus vient de l'intégration de la librairie Jquery. Easy met à dispostion quelques fonctionnalités telles que l'ouverture de popup, un menu déroulant, la gestion des erreurs formulaire, la rotation d'éléments pour slideshow... Ces fonctionnalités comportent une configuration par défaut avec des class prédéfinies ; lors du développement l'ajout d'une classe spécifique active l'événement (il est possible de surcharger la configuration par défaut avec ses propres paramètres).

L'utilisation de ce FrameWork est libre à condition de ne pas modifier certaines indications situées (copyright) dans les fichiers ; pour un usage commercial la license est pour le moment de 49$, elle sera bientôt de 149$.
