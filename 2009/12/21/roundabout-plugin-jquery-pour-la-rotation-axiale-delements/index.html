Roundabout, plugin jQuery pour la rotation axiale d'éléments

21/12/2009

Ce plugin jQuery ne paye pas de mine. Les premiers exemples présentés ne sont pas bien démontratifs des possibilités que celui-ci offre ; seulement si on fouine dans une liste en bas de page, on observe que certaines utilisations peuvent se révéler bien pratiques.

Le principe est assez simple, il s'agit d'une mise en forme plus animation d'une liste d'éléments (liste à puce). Ces éléments sont ensuite présentés comme dans un carrousel ; un clic sur l'élément suivant fait passer celui-ci au premier plan. Chacun des éléments tourne autour d'un axe dont la direction est définissable : horizontale, verticale...

Je trouve le principe sympa pour le choix d'éléments d'une liste déroulante dans un formulaire (voir illustration). J'imagine qu'il est possible d'animer à moindre frais, une petite liste d'images affichée simplement.

Le plugin a l'air quand même un peu lent, il manque un peu de fluidité ; cependant, le fonctionnement est assez agréable.
