TYPO3 : créer une version mobile de son site

03/04/2009

Comment réaliser une version mobile de son site

  • Un gabarit dédié
  • Une feuille de style spéciale
  • Une version modifiée (patch) de TemplaVoila
  • Une gabarit TemplaVoila spécifique à la version mobile
  • Un fichier PHP avec les fonction suivantes :
    • Détection de l'agent "mobile"
    • Instruction TemplaVoila pour afficher le gabarit dédié
  • Conditions TypoScript pour affichage d'un contenu particulier pour l'user agents "mobile"
