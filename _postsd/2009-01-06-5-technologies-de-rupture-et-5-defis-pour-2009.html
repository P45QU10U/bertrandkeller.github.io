SaaS, Cloud Computing, Virtualisation, Open Source et Social Computing sont selon le cabinet Saugatuck Technology les cinq technologies qui vont rythmer l'année 2009. Equilibrer les budgets, gérer l'évolution vers le cloud computing, gérer des infrastructures hybrides, consolider les fournisseurs et consolider les compétences tels sont les défis auxquels seront confrontés les DSI dans l'année qui vient.

Les 5 technologies phare de 2009

  • SaaS (Software as a Service)
  • Cloud Computing
  • Virtualisation
  • Open source
  • Social Computing

Les 5 défis majeurs des DSI pour 2009

  • Maîtriser les budgets IT
  • Evoluer vers le cloud computing
  • Gérer un environnement hybride
  • Faire face à la consolidation des fournisseurs et s'assurer de leur viabilité
  • Gérer les ressources
