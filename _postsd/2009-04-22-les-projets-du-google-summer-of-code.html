Le google été de code est un programme de financement de projets de développement open source pour étudiants.

Cette année des projets ont été sélectionnés pour les communautés Drupal et TYPO3.

Le verdict :

  • 18 projets pour Drupal => 90'000$
  • 4 projets pour TYPO3 => 20'000$

Les projets TYPO3

  • Code Kickstarter pour TYPO3 4.3 Extbase - dirigé par Jochen Rau
  • Collaborative Translation System - dirigé par Ingo Renner
  • Improving the FLOW3 Security Framework - dirigé par Robert Lemke
  • Versioning for the TYPO3 Content Repository - dirigé par Karsten Dambekalns
