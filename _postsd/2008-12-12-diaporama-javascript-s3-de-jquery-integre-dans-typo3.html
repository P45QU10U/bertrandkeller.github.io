Inspiré de Smooth Slideshow, S3 Slider propose d'enchaîner une série d'images sous forme de diaporama.

A vue d'oeil, ce plugin a l'air plutôt rapide. Son originalité consiste à afficher la légende de chaque image avec une animation. Je vous laisse vérifier par vous même les types d'animations de légendes disponibles.

Ce plugin est disponible sous TYPO3 grâce à l'extension : s3slider implementation for Typo3.
